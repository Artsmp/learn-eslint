# 在编辑器中集成 ESLint

> ESLint 可以和很多编辑器非常好的配合，可以在我们 dev 的时候就能够即使反馈 lint 信息，这一节我们就讲讲在编辑器中集成 ESLint 吧～

- 在 VScode 插件市场安装 ESLint 插件

![image-20220701075827099](https://tva1.sinaimg.cn/large/e6c9d24egy1h3r3jjpaxuj212j0u0jxu.jpg)
